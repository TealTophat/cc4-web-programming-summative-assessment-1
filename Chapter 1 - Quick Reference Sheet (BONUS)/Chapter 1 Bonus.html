HTML & CSS QUICK REFERENCE DOCUMENT

HTML TAGS


<head>

Contains meta information for the webpage.
				
			<head> 
			<title>HTML Quick Reference Guide</title>
			<link rel="stylesheet" href="mystyle.css">
			<meta charset="utf-8">
				</head>

<body>

Contains the visible content of the webpage.
			
			<body>
			<h1>Hello World!</h1>
				</body>

<h1>-<h6>

Creates text headings of different sizes.
			
			<h1>This is the biggest heading!</h1>
			<h6>This is the smallest heading!</h6>
Result:

This is the biggest heading!


This is the smallest heading!

<p>

Groups sentences and sections of text together.
			
			<p>
			Lorem ipsum dolor sit amet, consectetur adipiscing elit.
				</p>

<ol>

Makes an ordered list.
			
			<ol>
				<li>Eggs</li>
				<li>Potatoes</li>
				<li>Tomatoes</li>
				<li>Milk</li>
				</ol>
Result:

  1. Eggs
  2. Potatoes
  3. Tomatoes
  4. Milk

<ul>

Makes an unordered list.
			
			<ul>
				<li>Eggs</li>
				<li>Potatoes</li>
				<li>Tomatoes</li>
				<li>Milk</li>
				</ul>
Result:

  • Eggs
  • Potatoes
  • Tomatoes
  • Milk

<dl>

Makes a dictionary list.
			
			<dl>
				<dt>Vegetable</dt><dd>a plant or part of a plant used as food, such as a cabbage, potato, carrot, or bean.</dd>
				<dt>Herb</dt><dd>any plant with leaves, seeds, or flowers used for flavouring, food, medicine, or perfume.</dd>
				</dl>
Result:

Vegetable
a plant or part of a plant used as food, such as a cabbage, potato, carrot, or bean.
Herb
any plant with leaves, seeds, or flowers used for flavouring, food, medicine, or perfume.

<li>

Makes a list item.
			
			<li>Item 1</li>
			<li>Item 2</li>

<dt>

Makes a term for a dictionary list.
			
			<dt>Vegetable</dt>

<dd>

Makes a definition for a dictionary list.
			
			<dd>a plant or part of a plant used as food, such as a cabbage, potato, carrot, or bean.</dd>

<a>

Specifies a hyperlink reference (href) to a file.
			
			<a href="https://www.google.com">This is a link to Google.</a>
Result:
This is a link to Google.

<hr>

Creates a horizontal line.

<img>

Creates graphics on a webpage.
			
			<img src="dawn-8412840_1280.jpg">
Result:

Image taken from Pixabay.

<progress>

Displays a bar that depicts a numeric value within a specified range
		
		<progress value="12" max = "100">12%</progress>
Result:
12%

CSS PROPERTIES

color

Changes the color of text.
		
		<h3 style="color:red;">This is a red heading.</h3>
Result:

This is a red heading.


background-color

Changes the color of the background behind the element.
		
		<h3 style="background-color:yellow;">This is a yellow background..</h3>
Result:

This is a yellow background.


font-family

Changes the font typeface of text.
		
		<h3 style="font-family:Arial, Helvetica, sans-serif;">This heading is in the Arial font.</h3>
Result:

This heading is in the Arial font.


font-weight

Changes the thickness of text.
		
		<h3 style="font-weight:normal;>This heading has a normal thickness."</h3>
Result:

This heading has a normal thickness.


text-align

Changes the alignment of text.
		
		<h3 style="text-align:center;">This heading is in the center.</h3>
Result:

This heading is in the center.


letter-spacing

Changes the space between text characters.
		
		<h3 style="letter-spacing:3px;">This heading has a letter spacing of 3 pixels.</h3>
Result:

This heading has a letter spacing of 3 pixels.
